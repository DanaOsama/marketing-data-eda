# Marketing Truth Engine – EDA & Benchmark Generator

This repository prepares historical marketing data to power a plain-language AI insights engine.

### Data
[Global Ads Performance (Google, Meta, TikTok)](https://www.kaggle.com/datasets/nudratabbas/global-ads-performance-google-meta-tiktok/data)
The goal is to:
1. Split raw data into training and test sets
2. Generate historical performance benchmarks
3. Use those benchmarks to analyze new data and produce clear, business-friendly insights
